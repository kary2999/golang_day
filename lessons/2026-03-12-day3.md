# Day 3：函数 — 多返回值 & 错误处理

> 日期：2026-03-12

## Go 的多返回值

Go 函数可以同时返回多个值，这是它最有特色的设计之一。

```go
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("除数不能为零")
    }
    return a / b, nil
}

result, err := divide(10, 2)
if err != nil {
    fmt.Println("错误:", err)
} else {
    fmt.Println("结果:", result) // 5
}
```

## PHP 对比

```php
// PHP：用异常或返回数组
function divide($a, $b) {
    if ($b == 0) throw new Exception("除数不能为零");
    return $a / $b;
}

try {
    $result = divide(10, 2);
} catch (Exception $e) {
    echo "错误: " . $e->getMessage();
}
```

Go 不用 try/catch，错误是普通返回值，必须显式处理。

## 实用例子：读取配置

```go
func getConfig(key string) (string, bool) {
    config := map[string]string{"host": "localhost", "port": "8080"}
    val, ok := config[key]
    return val, ok
}

host, ok := getConfig("host")
if !ok {
    fmt.Println("配置不存在")
}
```

这种 `(value, ok)` 模式在 Go 中极为常见（map 取值、类型断言等）。

## 今日小结

> Go 用多返回值代替异常，错误不能被忽视 — 写起来啰嗦，但逻辑更清晰、更安全。
