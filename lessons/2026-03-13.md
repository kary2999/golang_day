# Day 4 · Struct 与方法

> 日期：2026-03-13

## 知识点：struct + 方法

Go 没有 class，用 **struct** 定义数据结构，用**方法**绑定行为。

### PHP（类）

```php
class User {
    public string $name;
    public int $age;

    public function __construct(string $name, int $age) {
        $this->name = $name;
        $this->age  = $age;
    }

    public function greet(): string {
        return "Hi, I'm {$this->name}";
    }
}

$u = new User("Alice", 30);
echo $u->greet(); // Hi, I'm Alice
```

### Go（struct + 方法）

```go
type User struct {
    Name string
    Age  int
}

// 方法：接收者 (u User)
func (u User) Greet() string {
    return "Hi, I'm " + u.Name
}

func main() {
    u := User{Name: "Alice", Age: 30}
    fmt.Println(u.Greet()) // Hi, I'm Alice
}
```

---

## 实用例子：修改字段用指针接收者

```go
// 值接收者：不修改原值
// 指针接收者：修改原值（类似 PHP 的引用传值）
func (u *User) Birthday() {
    u.Age++
}

u := User{Name: "Bob", Age: 25}
u.Birthday()
fmt.Println(u.Age) // 26
```

> **规律**：只读用值接收者 `(u User)`，需要修改用指针接收者 `(u *User)`

---

## 今日小结

Go 的 struct + 方法 ≈ PHP 的 class，但更轻量；指针接收者让你控制"改没改原对象"，比 PHP 的引用更显式。
