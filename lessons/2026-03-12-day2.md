# Day 2：控制流（if / for / switch）

> 日期：2026-03-12

## if 语句

Go 的 `if` 条件不需要括号，但大括号必须有。

**Go:**
```go
if x > 10 {
    fmt.Println("大")
} else {
    fmt.Println("小")
}
```

**PHP:**
```php
if ($x > 10) {
    echo "大";
} else {
    echo "小";
}
```

> 💡 Go 的 `if` 还支持初始化语句：
> ```go
> if err := doSomething(); err != nil {
>     log.Fatal(err)
> }
> ```

---

## for 循环

Go **只有 `for`**，没有 `while`/`do while`。

**Go（当 while 用）:**
```go
i := 0
for i < 5 {
    i++
}
```

**PHP:**
```php
$i = 0;
while ($i < 5) {
    $i++;
}
```

---

## switch 语句

Go 的 `switch` 不需要写 `break`，默认不穿透。

**Go:**
```go
switch status {
case 1:
    fmt.Println("成功")
case 0:
    fmt.Println("失败")
default:
    fmt.Println("未知")
}
```

**PHP:**
```php
switch ($status) {
    case 1:
        echo "成功";
        break;
    case 0:
        echo "失败";
        break;
    default:
        echo "未知";
}
```

---

## 实用例子：HTTP 状态码处理

```go
func describeStatus(code int) string {
    switch {
    case code >= 200 && code < 300:
        return "成功"
    case code >= 400 && code < 500:
        return "客户端错误"
    case code >= 500:
        return "服务器错误"
    default:
        return "未知状态"
    }
}
```

Go 的 `switch` 不带表达式时，相当于 `if-else if` 链，非常优雅。

## 今日小结

> Go 控制流更简洁：`if` 不加括号、`for` 一统天下、`switch` 自带 `break`——少写代码，少犯错。
